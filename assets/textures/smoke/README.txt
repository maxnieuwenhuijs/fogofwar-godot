Zwartkruit-rook textures (musket + kanon)
==========================================
Drop hier rook-afbeeldingen. Twee soorten werken allebei:
- PNG met transparantie (alpha), of
- AI-output 'isolated on solid black background': zwart wordt bij het
  laden automatisch omgezet naar transparantie (helderheid = dekking).

Elke rookwolk (loop + inslag, musket en kanon) kiest een willekeurige
texture uit deze map. Map leeg = grijze bol-wolkjes (fallback).

Tunen in de Model-tuner: rook-aantal, rook-maat, rook-groei (hoe ver de
wolk uitzet), rook-duur. Testknoppen: rook (musket) / rook (kanon).
