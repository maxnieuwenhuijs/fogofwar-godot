Bloedspetter-textures
=====================
Drop hier PNG-bestanden (met transparantie/alpha) van bloedspetters of
-plassen. Het spel pakt ze automatisch op: elke bloedplas kiest een
willekeurige texture uit deze map. Map leeg = simpele rode schijfjes.

Tips:
- Vierkant werkt het best (bv. 256x256 of 512x512).
- Rood op transparante achtergrond; artillerie-roet wordt donker getint.
- Elke bestandsnaam is goed (splat1.png, pool_a.png, ...).
